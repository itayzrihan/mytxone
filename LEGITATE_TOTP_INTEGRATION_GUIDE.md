## ğŸ¯ Overview

This guide documents the complete implementation of **TOTP (Time-based One-Time Password)** two-factor authentication using **Legitate's Simple TOTP** service integrated into the mytx.one platform.

### Key Features

- âœ… **Mandatory 2FA** for all new registrations
- âœ… **Popup-based setup** (like PayPal/Google OAuth)
- âœ… **Dynamic code verification** with smart UI
- âœ… **Encrypted secret storage** (AES-256-GCM)
- âœ… **Rate limiting** (5 attempts per 15 minutes)
- âœ… **Registration token system** for tracking setups
- âœ… **TOTP standard compliance** (RFC 6238)
- âœ… **Beautiful confirmation pages**
- âœ… **Cross-domain callback support**

### Technology Stack

- **Backend:** Next.js 15 API Routes, TypeScript
- **Database:** PostgreSQL with Drizzle ORM
- **Authentication:** NextAuth.js
- **2FA Provider:** Legitate Simple TOTP
- **Encryption:** AES-256-GCM (Node.js crypto)
- **Rate Limiting:** Redis (Upstash)
- **UI:** React, TailwindCSS, Radix UI
