# 🎆 עדכון דף הנחיתה בהיפנוזה - עברית מלאה עם RTL

## ✅ שינויים שבוצעו

### 1. **תמיכה RTL מלאה** 📱
- הוסיף `dir="rtl" lang="he"` לכל דף
- כל הטקסט כעת יוצג מימין לשמאל
- פריסה הלוגיקה מתהפכת אוטומטית

### 2. **דף נחיתה - תרגום מלא** (/hypno)

#### כותרת ובעלות
```
להפנט את היקום
Hypnotize the Universe
מאת: איתי זריהן
```

#### תיאור הספר (עמודה שמאל)
- **כותרת**: "פתחו את עולמכם לאפשרויות אינסופיות"
- **תיאור**: גלו את העולם הנסתר, למדו הסודות של ההיפנוזה
- **3 סעיפים עם סמלים**:
  1. הבנת ההיפנוזה - "היפנוזה אינה מה שאתם חושבים שהיא"
  2. ידע אתי ואחראי - "למדו להגן על עצמכם"
  3. שנו את המציאות שלכם - "אם לא תבינו את הכוחות הללו..."

#### טופס (עמודה ימין)
- **כותרת**: "קבלו את הספר החינם שלכם"
- **שדות**:
  - שם מלא *
  - כתובת אימייל *
  - מספר טלפון * (ללא +972, רק `050-000-0000`)
  
- **תיבות בחירה**:
  - "אנחנו רוצים לקבל עדכוני שיווק ותוכן אקסקלוסיבי"
  - "אנחנו מסכימים לקבל ידע ושיטות בהיפנוזה"
  
- **כפתור**: "פתחו את הספר החינם 🔓"
- **הודעות**: 
  - הצלחה: "✨ הספר שלכם מוכן!"
  - שגיאה: תצוגת שגיאות ברורה

#### תוכן תחתון
- **3 יתרונות**:
  1. 100% חינם - ללא תשלומים נסתרים
  2. גישה מיידית - קבלו בתוך דקות לאימייל שלכם
  3. שינוי חיים - שנו את ההבנה על התודעה

- **ציטוט מהספר**:
  - "כשיש לכם את הידע הזה, העולם משתנה. לא המציאות משתנה, אלא תפיסתכם שלה."
  - ייחוס: איתי זריהן

### 3. **דף הורדה - תרגום מלא** (/hypno/ebook1)

#### הודעת ברכה
- **כותרת**: "🎉 ברוכים הבאים, מחפשי הידע"
- **תת-כותרת**: "המסע שלכם לתודעה מתחיל עכשיו"
- **הודעה**: "הספר החינם שלכם מוכן להורדה"

#### כרטיס הספר
- **שם**: להפנט את היקום
- **תרגום**: Hypnotize the Universe
- **מחבר**: מאת איתי זריהן
- **תיאור**: גלו את העולם הנסתר, למדו את סודות ההיפנוזה
- **פרטים**:
  - פורמט: PDF
  - שפה: עברית ואנגלית
  - גישה: לכל החיים

#### כפתור הורדה
- "הורדו את הספר עכשיו"
- סטטוסים:
  - טוען: "בהכנה להורדה..."
  - הושלם: "ההורדה התחילה! ✨"

#### הודעות מידע
1. **📧 משלוח בדוא"ל**: "הספר נשלח גם לכתובת הדוא"ל שלכם כדי שתוכלו לשמור עליו"
2. **🔐 הפרטיות שלכם**: "אנחנו מכבדים את התודעה שלכם. המידע שלכם ישמש רק למטרות שהסכמתם להן"
3. **✨ המסע שלכם**: "פתחו את הספר כאשר אתם מוכנים. אין שום חיפזון. הידע יחכה לכם"

#### סיום
- **שאלה**: "יש לכם שאלות או רוצים לשתף את החוויה שלכם?"
- **CTA**: "יצרו קשר ←"

## 📝 שינויים לשון

### משהו לקוראים בודדים (זכר) → בודדים (שונים)

**דוגמאות**:
- "פתח" → "פתחו"
- "גלה" → "גלו"
- "שנה" → "שנו"
- "אתה חושב" → "אתם חושבים"
- "אני רוצה" → "אנחנו רוצים"
- "אני מסכים" → "אנחנו מסכימים"
- "יש לך" → "יש לכם"
- "רוצה" → "רוצים"
- "קבל" → "קבלו"
- "כל מי שיודע" → "כל מי שיודעים"
