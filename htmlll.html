Liquid Glass Effect

Scroll down to see content morph behind the fixed glass elements.

How It Works

This effect is achieved without using any SVG filters. Instead, it relies on multiple layers of the CSS backdrop-filter property. Each layer has a slightly different blur, saturation, and brightness value, creating a gradient from a clear, saturated center to a distorted, bright edge, mimicking a real glass lens.

Layered Filters

The core bubble has three layers. The outer layer has the highest blur, the middle layer is clearer, and the center has zero blur with high saturation.

Dynamic Shine

A touch of JavaScript adds a radial gradient that follows your mouse, creating a realistic lighting effect and enhancing the 3D illusion.

Cross-Browser

Designed to be fully compatible with modern browsers like Chrome and Safari on both desktop and mobile platforms.

Any Shape

The class-based structure allows this effect to be applied to any element shape by simply adjusting the border-radius.
